### 3. 密码可见性切换
**文件**: `pages/index/index.wxml` + `pages/index/index.wxss` + `pages/index/index.js`

**WXML**:
```wxml
<view class="input-with-icon">
  <input type="{{showPassword ? 'text' : 'password'}}" ... />
  <text class="pwd-toggle" bindtap="onTogglePassword">
    {{showPassword ? '👁️' : '👁️‍🗨️'}}
  </text>
</view>
```

**WXSS**:
- `.input-with-icon`: flex 容器
- `.field-input-icon`: 输入框（右留空 60rpx 给图标）
- `.pwd-toggle`: 绝对定位，右侧眼睛图标

**JS**:
```js
data: { showPassword: false }
onTogglePassword() { this.setData({ showPassword: !this.data.showPassword }) }
```
