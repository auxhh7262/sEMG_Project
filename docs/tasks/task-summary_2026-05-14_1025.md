1. **ADC参考5V正确**：mean=5206.6 → DCbias=1589mV ≈ 1.5V传感器基准
2. **AC信号极弱**：峰峰值384 counts(117mV)，正常应~3000mV → 只有正常的3.9%
3. **固件算法无误**：RMS/MDF计算逻辑正确，问题在传感器信号本身
4. **放大倍数更正**：1000倍（不是1500倍），MEMORY.md已更新
