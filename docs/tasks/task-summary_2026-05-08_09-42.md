# sEMG 实时监测双曲线显示 — 2026-05-08 09:42

## 目标

在 realtime 页面的主示波器 Canvas 上同时叠加显示两条参考曲线：
- **群体参考曲线**（B区）：橙色虚线
- **个人曲线**（A区）：绿色细虚线
固件生成个人曲线后，小程序存储并传递给 realtime 页面渲染。

## 改动摘要

| 文件 | 改动 |
|------|------|
| `pages/realtime/index.js` | `_evalCurve()` 五次多项式展开 + `_loadCurves()` 加载缓存曲线 + `_drawWave()` 叠加双虚线 |
| `pages/calibrate/index.js` | `curve_gen_result` 响应中同步存储 `personal_curve` 和 `group_curve` 到 Storage |
| `pages/realtime/index.wxss` | 图例颜色对齐（个人曲线→绿色、B区→橙色）+ TabBar 底部留白 |

### 渲染逻辑

```
_drawWave() 每帧执行：
  1. 清空 Canvas
  2. 叠加群体参考曲线（B区）：橙色 [6,4] 虚线
  3. 叠加个人曲线（A区）：绿色  [3,3] 虚线
  4. 绘制网格线
  5. 绘制 RMS 实时实线（绿色实线，最顶层）
```

## 数据流

```
calibrate 校准成功
  → 用户选曲线 → gen_personal_curve
    → 固件返回 curve_gen_result { ok, coef, baseline_rms, baseline_mdf }
      → 小程序存 personal_curve + group_curve 到 Storage
        → _jumpToRealtime()
          → realtime Tab onShow → _loadCurves()
            → Canvas 每帧绘制双虚线
```
