### 2. 配网说明文字优化
**文件**: `pages/index/index.wxml`

- 第1步：点击「扫描设备」，搜索附近的 sEMG 手环（需开启手机蓝牙）
- 第2步：连接成功后，在上方显示设备名称：`<hl>{{deviceName || '待连接'}}</hl>`
- 第3步：输入 WiFi 名称和密码，点击「配置网络」发送凭证给设备
- 第4步：配置成功后，设备自动获取 IP 地址（显示在按钮上方）
- 第5步：配网完成后，切换到底部「实时监测」标签查看肌电数据
- 警告：仅支持 2.4GHz 网络；点击「重置网络」可清除配网信息

### 3. 密码显示切换
**文件**: `pages/index/index.wxml` + `pages/index/index.wxss` + `pages/index/index.js`

- 图标改为文字「显示」/「隐藏」（更可靠，不依赖emoji渲染）
- 逻辑：点「隐藏」→ 密码显示明文；点「显示」→ 密码显示星号
- `catchtap` 防止事件冒泡
