# sEMG小程序设置页合并 - 2026-05-09

## 目标
删除wifi-config独立页面，将BLE配网逻辑内联到设置Tab（index页），简化页面结构和用户操作路径。

## 执行
1. **index.wxml** - 重写：移除纯跳板按钮，内联BLE配网（扫描→连接→输入SSID/密码→发送→等待IP→显示结果），暗色主题
2. **index.js** - 重写：从wifi-config迁移BLE状态机（STEP枚举7状态）、bleClient回调、WiFi凭证发送、IP缓存Storage，移除TCP连接和串口调试面板
3. **index.wxss** - 重写：暗色主题（#0f0f0f背景），状态badge配色，表单锁定态
4. **index.json** - 更新导航栏标题
5. **app.json** - 删除wifi-config路由，pages从5项减至4项

## 兼容性
- realtime/index.js 通过 `wx.getStorageSync('device_ip')` 读取IP → index.js 仍写Storage，兼容
- calibrate/index.js 无wifi-config引用，不受影响
- bleClient.js v2.2 的 `onIpReceived()` 和 `sendWifiConfig()` 直接复用
