# Task Summary: sEMG代码审查Bug修复批次2

## Objective
继续修复代码审查发现的剩余4项🟡🟢级问题。

## Key Reasoning
1. **DHCP阻塞**是唯一影响实时性的Bug：WiFi连接成功后的5秒`while+delay(100)`阻塞期间，10Hz调度、BLE poll、WebSocket全停。改为非阻塞状态机（每tick检查IP，5秒超时）。
2. **堆分配**在RAM 69.8%的嵌入式环境下风险大：`new uint16_t[3000]`×2 = 12KB堆分配可能碎片化。改为栈上256点分批写入（1KB栈，12批）。
3. **校准f/q/a全0**：校准阶段没有校准数据，fatigue/quality/activation确实无法计算。加`"mode":"calib"`标记让小程序区分处理。
4. **RMS归一化**：已有`_restRms/_maxRms`校准数据但未用于归一化，改用`(rms-rest)/(max-rest)`，无校准时回退硬编码。
