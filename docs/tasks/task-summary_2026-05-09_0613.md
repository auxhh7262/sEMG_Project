## 设计思路确认

### 建库数据流

1. **固件端**: `_handleDbFeatureState()` 每100ms写入特征点到B区
2. **固件→小程序**: 10Hz发送实时RMS/MDF数据
3. **小程序端**: `_dbOnFeatureSample()` 接收并通过Canvas绘制
4. **原始波形**: 固件内部1kHz采集，存入Flash，B区

### 波形显示策略

- 显示最近50个数据点 (5秒)
- 双线图: RMS(橙色) + MDF(蓝色)
- 每10个样本绘制一次，避免Canvas绘制过于频繁
- 网格背景便于阅读
