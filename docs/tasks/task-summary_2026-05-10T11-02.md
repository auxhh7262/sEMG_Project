## 问题诊断

**编译失败原因**：WXML 不支持 `!(bleConnected || deviceIp)` 这种带括号的否定表达式。

**修复方法**：应用 De Morgan 定律
```
!(a || b) → !a && !b
!(bleConnected || deviceIp) → !bleConnected && !deviceIp
```

## 完成的修改

## WXML 表达式兼容性提醒
微信小程序 WXML 模板表达式限制：
- ❌ 不支持 `!(a || b)` 带括号的复杂否定
- ✅ 支持 `!a && !b` 分解形式
- ✅ 支持 `a || b`、`a && b`、`!a` 等简单表达式
