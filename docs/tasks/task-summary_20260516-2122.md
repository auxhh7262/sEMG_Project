# 建库页面UI修复

## 目标
修复建库页面(dbcollect)的UI问题：删除不需要的canvas波形图、缩小数值字体

## 已完成修复
1. **删除canvas波形画布**：移除 db-waveform-wrapper 整块（canvas元素 + — RMS / — MDF 图例）
2. **数值字体缩小**：`.live-value` 和 `.countdown-val` 从 28px → 20px

## 未解决：数据全零+倒计时不动
用户截图显示 RMS=0.00, MDF=0.0, 剩余时间=300 不变。

### 数据链路分析
小程序 startDatabaseCollection → confirmDbStart → _startDbFeatureCollection:
1. setData({dbPhase:'collecting', phase:'dbcollect'})
2. 发送 {"cmd":"start_db_feature",...} 给固件
3. 注册 _dbMsgListener 监听 WebSocket 消息
4. 收到 {r:rms,m:mdf} 后调用 _dbOnFeatureSample() 更新界面

### 固件端
- AppController::_handleDbFeatureState() 在 ST_DB_FEATURE 状态下:
  - 调用 _signalProc->update() 获取 rms/mdf
  - 调用 _netMgr->sendData(rms, mdf, ...) 发送
  - **但 sendData() 有 `if (!_tcpStreaming) return;` 守卫**
  - **且 `if (rms > 0.0f)` 才发送** — 传感器悬空时不发

### 可能根因
1. 固件未成功转入 ST_DB_FEATURE 状态（命令失败）
2. _tcpStreaming 未被设为 true
3. 传感器未贴好/无信号（rms=0不发送）
4. 需要串口日志确认

### 下一步
