### 2. 移除"前往配网"按钮 ✅

**修改文件：**
- `pages/realtime/index.wxml` - 移除按钮
- `pages/realtime/index.js` - 移除 `goToNetwork` 方法

**修改前：**
```xml
<text class="dc-desc">请先在配网Tab连接设备</text>
<button class="btn btn-primary" bindtap="goToNetwork">前往配网</button>
```

**修改后：**
```xml
<text class="dc-desc">请点击下方「配网」Tab 连接设备</text>
<!-- 无按钮，用户直接点击 TabBar -->
```

**原因：** 减小页面间耦合，用户可直接点击 TabBar 切换页面

### 5. 建库流程校准信息 ✅（已确认正确）

**确认结果：** 建库流程已包含校准信息

当前实现：
1. 建库开始前 3 秒静息采集 → 计算 `rest_rms_mv`, `rest_mdf_hz`
2. 运动采集期间 → 计算 `max_rms_mv`, `max_mdf_hz`
3. `save_record` 命令包含完整校准数据：
   ```json
   {
     "cmd": "save_record",
     "name": "建库名称",
     "age": 25,
     "gender": 1,
     "handedness": 1,
     "rest_rms_mv": 0.1234,
     "max_rms_mv": 0.5678,
     "rest_mdf_hz": 85.6,
     "max_mdf_hz": 72.3,
     "rms_sequence": [...],
     "mdf_sequence": [...]
   }
   ```

**结论：** 无需修改，当前实现正确。每个建库记录自带独立的校准基线。
