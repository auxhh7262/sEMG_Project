# sEMG 固件曲线库实施 — 2026-05-08 08:43

## 目标
固件端 B区曲线库 + A区个人曲线管理全链路实现，为小程序曲线匹配与个人曲线生成提供固件接口。

## 已完成

### 1. StorageArch.h — 结构体对齐
- `AZone_Sector_t`：curve_coef[3→5] + curve_baseline_rms + curve_baseline_mdf + curve_status(新增1字节) + curve_reserved调整 → 总大小 4096 字节 ✅
- `BZone_CurveLibrary_t`：63条曲线（64→63确保fit 32KB），每条新增 handedness_hint/contributing_slots[8]/active_slot_count/curve_status/reserved_curve[1]；移除严格 size 断言，保留边界检查 ✅

### 2. StorageManager.h — 类型定义
- `PersonalCalibData_t`：对齐 StorageArch，curve_coef[5] + curve_baseline_rms + curve_baseline_mdf + curve_status
- 新增 `BZone_CurveEntry_t`（小程序通信用结构体，12字段）
- 新增方法声明：`BZone_ListCurves` / `BZone_GetCurve` / `GeneratePersonalCurve`

### 3. StorageManager.cpp — 接口实现
- `Init()` / `UpdatePersonalCalib()`：处理新字段（5系数 + baseline + status）
- `BZone_ListCurves(gender, handedness, age)`：从 Flash 读曲线库，支持三项条件过滤，返回最多20条
- `BZone_GetCurve(curveId)`：按ID查单条
- `GeneratePersonalCurve(bcurve_id, personal_baseline_rms, personal_baseline_mdf)`：方案A平移算法，约束±50%，写A区Flash

### 4. Globals.h — 新命令枚举
- `CMD_LIST_CURVES` / `CMD_GEN_PERSONAL_CURVE` / `CMD_GET_CURVE`

### 5. ProtocolHandler.cpp — JSON路由
- `list_curves` → handleListCurves
- `gen_personal_curve` → handleGenPersonalCurve
- `get_curve` → handleGetCurve

### 6. AppController.h/cpp — handler实现
- `handleListCurves`：返回 curve_list JSON（最多20条，含系数/基准/条件/质量）
- `handleGenPersonalCurve`：返回 curve_gen_result JSON（ok + curve_id + baseline）
- `handleGetCurve`：返回 curve_detail JSON（单条完整详情）
