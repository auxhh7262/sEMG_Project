# sEMG 手环 — 硬件说明文档

> 版本：v1.3 | 更新：2026-05-27 | 状态：优化章节结构，消除重复

---

## 1. 系统概览

本项目为基于**表面肌电（sEMG）**的肌肉疲劳监测手环，通过干电极传感器采集肌肉电信号，经 Arduino UNO R4 WiFi（RA4M1）处理后以 WebSocket 方式输出疲劳度和激活度数据。

```
人体肌肉
   ↓ 肌电信号（±1.5mV）
干电极传感器（放大1000×，输出0~3.0V）
   ↓ 模拟电压
Arduino UNO R4 WiFi（RA4M1 14-bit ADC）
   ↓ 数字信号处理
W25Q128FVSG SPI Flash（16MB，数据持久化）
   ↓
WiFi WebSocket → 微信小程序
```

## 2. 核心器件

### 2.1 主控：Arduino UNO R4 WiFi

| 参数 | 值 |
|------|-----|
| MCU | RA4M1 (ARM Cortex-M4F) @ 48 MHz |
| SRAM | 32 KB |
| Flash | 256 KB |
| Data Flash | 8 KB（存 WiFi 凭证） |
| ADC | 14-bit SAR，0~5V，0.305 mV/LSB |
| WiFi | ESP32-S3-MINI（板载，固件通过 UART 通信） |
| 工作电压 | 5V（USB）/ 7~12V（DC） |

**⚠️ SPI 电平注意：** RA4M1 的 SPI 引脚（D10~D13）为 **5V 电平**，外接 3.3V 器件（如 W25Q128FVSG）**必须**加电平转换模块，严禁 5V 直连。

### 2.2 肌电传感器：思知瑞干电极 sEMG 模块

| 参数 | 值 | 说明 |
|------|-----|------|
| 检测范围 | ±1.5 mV（差分） | 肌肉放松时约 ±0.01~0.1 mV |
| 增益 | 约 1000~1500× | 输出 0~3.0V |
| 输出范围 | 0 ~ 3.0 V | 以 1.5V 为基准中心 |
| 有效频带 | 10 ~ 500 Hz | 制造商标称，实际主要能量 20~250 Hz |
| 供电电压 | 3.3 ~ 5.5 V | 推荐 5V |
| 供电电流 | ≥ 20 mA | — |
| 电极接口 | PJ-342 干电极 | 50cm 导联线 |
| 模块接口 | XH2.54-3P | 红（5V）、黑（GND）、黄（信号 A0） |

**信号特点：**

| 状态 | RMS（传感器输出） | 对应 ADC 读数 |
|------|------------------|--------------|
| 放松 | 10~100 mV | 静息约 1.5V±100mV |
| 最大随意收缩（MVC） | 1~2 V | 可达 2.5V |
| 肌肉疲劳时 | MDF 下降 | 频谱向低频偏移 |

```
电压(mV) = analogRead(A0) / 16383 × 5000
RMS(mV) = (电压 - 1500mV) / 放大倍数
```

### 2.3 外置存储器：W25Q128FVSG

| 参数 | 值 | 说明 |
|------|-----|------|
| 型号 | W25Q128FVSG | Winbond（与 W25Q128JVSIQ ID不兼容） |
| 容量 | 16 MB（128 Mbit） | — |
| 工作电压 | **2.7 ~ 3.6 V** | ⚠️ 严禁 5V 直连！ |
| 接口 | SPI（Mode 3） | 最高 104 MHz，实际使用 1 MHz |
| 扇区大小 | 4 KB | — |
| 块大小 | 64 KB | — |
| 擦除寿命 | >100,000 次 | — |
| JEDEC ID | 0xEF4018（M=0xEF, T=0x40, C=0x18） | — |

**⚠️ 电平转换：** 必须通过 BSS138 MOSFET 双向电平转换模块连接（详见第 3.2 节）。若曾 5V 直连，芯片可能已损坏，需更换全新 W25Q128FVSG。

## 3. 电路连接

### 3.1 传感器接口

```
sEMG 模块           Arduino UNO R4 WiFi
---------           -------------------
  5V（红）    ←    5V（给传感器供电）
  GND（黑）   ←    GND
  AOUT（黄）  →    A0（ADC 输入）
```

### 3.2 SPI Flash 接口（BSS138 电平转换）

使用 **4 路 BSS138 MOSFET 双向电平转换模块**：

```
W25Q128FVSG              电平转换模块 (BSS138)              UNO R4 WiFi
------------------      ---------------------------      ---------------
  VCC            —     LV(3.3V电源)           —     HV(5V电源)
               ↑   Arduino 3V3供电      ↑   Arduino 5V供电
  GND            —     GND                     —     GND
  /CS            ←     LV2 ── HV2              ←     D10 (CS)
  DI (MOSI)      ←     LV3 ── HV3              ←     D11 (MOSI)
  DO (MISO)      →     LV1 ── HV1              →     D12 (MISO)
  CLK (SCK)      ←     LV4 ── HV4              ←     D13 (SCK)
```

**接线要点：**
- **LV 电源**：BSS138 LV 引脚接 Arduino 3.3V 输出（低电压侧电源）
- **HV 电源**：BSS138 HV 引脚接 Arduino 5V 输出（高电压侧电源）
- **Flash VCC**：接 3.3V 电源（与 BSS138 LV 同源）
- **GND**：Flash GND、BSS138 GND、Arduino GND 三者共地
- BSS138 双向自动感知，无需区分方向；箭头仅表示 SPI 信号流向

## 4. 信号处理链

```
ADC Raw (0~16383)
    ↓ × (5000/16383) mV
传感器输出电压 (0~3000mV)
    ↓ - 1500mV（中心偏置去除）
双向 sEMG 信号 (±1500mV)
    ↓ ÷ 1000（传感器增益还原）
肌肉电势差 (±1.5mV)
    ↓ 直流去除（高通滤波 + EMA）
交流 sEMG (0~±1.5mV)
    ↓ FFT → 功率谱
    ↓ 积分 10~250Hz
MDF（中位频率）Hz
    ↓ EMA 平滑
MDF_filtered
    ↓ 1 - (MDF / restMDF)
疲劳度 0~100%
```

**采样率：**

| 阶段 | 频率 | 说明 |
|------|------|------|
| 原始 ADC 中断采样 | ~500 Hz（实测） | gpt_timer 中断 |
| FFT 窗口 | 256 样本 | 约 512ms 窗口（@500Hz） |
| RMS/MDF 输出 | 10 Hz | 主循环，每 100ms 一次 |

**校准参数：**

| 阶段 | 时长 | 说明 |
|------|------|------|
| REST 校准 | 10 秒（丢弃前 3 秒） | 消除 EMA 残留 |
| MAX 校准 | 15 秒（上升沿检测） | 锁定真收缩期 |

## 5. WiFi 与配网

### 5.1 凭证存储

WiFi SSID 和密码存储在 RA4M1 **Data Flash（EEPROM）** 区域（8KB），通过 PlatformIO EEPROM 库访问。

### 5.2 配网流程（小程序）

1. 小程序扫描设备 AP（设备作为热点）
2. 小程序发送 SSID + 密码到设备
3. 设备切换为 Station 模式连接目标 WiFi
4. 设备向小程序回报 IP 地址
5. 小程序切换到 TCP 连接模式

| 区域 | 地址范围 | 大小 | 用途 |

- v3.9.11 及以后版本含 `peak_rms_mv`（float，峰值 RMS）

## 8. 已知限制与注意事项

1. **电平转换** — W25Q128FVSG 必须通过 BSS138 模块连接（LV=3.3V，HV=5V），严禁 5V 直连。MISO 方向为 Flash→Arduino，接 LV1/HV1 通道。
2. **Flash 芯片不兼容** — W25Q128JVSIQ 的 JEDEC ID 读到 0x0B（异常，应为 0xEF），本系统只支持 W25Q128FVSG（JEDEC ID: 0xEF4018）。若曾 5V 直连，需更换全新芯片。
3. **ADC 参考电压** — 使用板载 5V 参考，可改用 3.3V 参考以提高 sEMG 信号分辨率。
4. **传感器贴附** — 干电极需与肌肉纤维方向一致；参考电极应贴在肌肉附近但非目标区域；皮肤干燥或接触不良会导致 MDF 不稳定。
5. **WiFi 距离** — 设备 WiFi 信号较弱，测试时手机与设备保持 1 米以内。
6. **曲线拟合** — 离线完成（电脑端 MATLAB/Python），拟合后写回 B 区曲线库，A 区存储匹配结果。
7. **校准电极要求** — 必须贴附电极后才能校准，否则 RMS ratio 接近 1.0（仅 DC 偏置，无 AC 肌电成分）。
